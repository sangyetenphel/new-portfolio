Smart Weather App

I build this app after I completed by Web Programming course with Python and JavaScript from Harvard University.

It shows the current website of the city name you entered and recommends what to wear based on the current location of that place.

Fetched live weather data from the OpenWeather API.

Customers had the ability to view all the cities in the world from a JSON file.

Technologies:
  • - Python
  • - Django
  • - JavaScript
  • - JSON file
  • - Heroku
  • - OpenWeather API
