Furever Precious

I build this website for as an E-commerce website to sell pet products online with the Stripe payment system for secure payment.

User can sign up and login to save their information or buy products without logging in as well.

Users can then add products to their carts and proceed to checkout when desired.

Technologies:
  • - Python
  • - Django
  • - JavaScript
  • - Postgres
  • - Stripe API

Furever Precious sign up page.
Login to be greeted with your user details.
Signed in users can leave products reviews as well.
Add products to the cart.
View all the products in your cart.
Finally proceed to checkout.
